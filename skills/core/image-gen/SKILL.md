# Image Generation

Generate images from text prompts using AI (DALL-E).

## Tools

- `generate_image` — Create an image from a text prompt

## When to Use

- User asks you to "create an image", "draw", "generate a picture", "make a visual"
- User asks for a screenshot (use browse tool instead)
- When a visual response would be more helpful than text

## Rules

- Always generate the image immediately when asked — don't ask for confirmation
- Use descriptive, detailed prompts for better results
- Tell the user what you generated after sending
